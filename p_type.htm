Circular Sawing Machines:  P-65A, P-100A, P-150A

　

Model No. Standard Features Specifications
　

P-65A

　

  • Length index device, drive by servo motor and ball screw.

  • Hydraulic clamped front & vertical vises.

  • Floating shuttle device.

  • Sorting chute

  • Disk brake device for eliminating transmission backlash.

  • Chip brush.

  • Screw type chip conveyor

  • Variable blade speed controlled by inverter

  • Automatic loading table (5M) - Only suitable for cutting round bar

  • Saw blade 2 pcs

  • Brake blade 2 pcs

  • Transmission belt 2 pcs

  • Cutting mist oil 5 gallon

  • Tools box with tools

Optional Accessories:

  • Chain type chip conveyor

  • Air filter

  • Air compressor (vertical screw type)

  • Air compressor (horizontal belt type)

  • Variable vise pressure

  • Machine with CE regulation

  • Automatic loading table (5M) suitable for cutting square bar
